Map Method

11/1/2014

This week focus on ruby algorithms and learning more about this programming language Ruby. Today I want to talk you about very powerful Enumerable method map. What map method does is goes through every element of collection and preforms a block of code that you give. The map method returns the resolve of that code block as a new array. So let’s take a look at our array from last week called list_favorite_cities which is a list of favorite cities.

list_favorite_cities = ["New York", "San Franciso", "Chicago", "Dallas", "Los Angeles"]

Let’s say we want to make a welcome message for each city. You can do that with by calling the map method on like the following.

welcome = list_favorite_cities.map do |city| "Welcome to #{city}." end

puts welcome

puts list_favorite_cities

What will happen is welcome variable will be set to the new array with all the new elements. So, not only does map perform block of code it also doesn’t change the original collection. There more powerful things you can do with this method but that enough for now.
